# .ai/deep-interview폴더에서 SKILL.md 작업중 산출물을 제시한 후 반드시 확인을 요청을 받으면 아래 규칙을 진행한다.

## workflow 생성 규칙

- workflow 만들기 요청 시 .ai/deep-interview/SKILL.md 방식으로 질문 먼저하기.
- workflow 실행 요청 시 해당 작업할 폴더 혹은 파일에 주석으로 작업할 목록을 적은 후 작업 진행시작 할건지 물어보기.
- doc/dev-a-b/폴더에 '주차-#번째작업-workflow.md'형식으로 만들어.
- workflow 작성 후 작업이 진행되고 나면 오류가 없는지 디버깅 및 테스트 진행.

## report 생성 규칙

- 작업한 내용을 정리해서 doc/dev-a-b/폴더에 '주차-#번째작업-report.md' 형식으로 만들어.
