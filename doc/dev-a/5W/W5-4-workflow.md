# W5-4 Workflow — 예약 변경 화면 지난 시간 슬롯 비활성화

> **작성일**: 2026-03-20
> **브랜치**: `feature/reservation`
> **목표**: 예약 변경 화면에서 오늘 날짜 선택 시 현재 시각 이전 시간 슬롯을 회색 글자로 표시 + 선택 불가 처리

---

## 전체 흐름

```
사용자가 오늘 날짜 선택
  → datePicker onChange 트리거
  → booked-slots API 호출
  → 응답 수신 후 슬롯 렌더링
      ├─ booked 슬롯 → 회색 + "(예약불가)" + disabled
      ├─ 지난 시간 슬롯 → 회색 글자 + disabled (텍스트 변경 없음)
      └─ 선택 가능 슬롯 → 정상 표시
```
