# .ai/deep-interview폴더에서 SKILL.md 작업중 산출물을 제시한 후 반드시 확인을 요청을 받으면 아래 규칙을 진행한다.

## workflow 생성 규칙

- workflow 만들기 요청 시 .ai/deep-interview/SKILL.md 방식으로 질문 먼저하기.
- doc/dev-a/폴더에 '주차-#번째작업-workflow.md'형식으로 만들어.
- workflow는 전체 흐름과 인터뷰 결과, ui mockup, 작업 목록(주석x), 작업 진행내용, 테스트 진행, 완료기준 형태로 보여줘야해.
- workflow 작성 후 작업할때 코드에 상세 주석을 적고 진행되고 나면 오류가 없는지 디버깅 및 테스트 진행.

## report 생성 규칙

- 작업한 내용을 정리해서 doc/dev-a/폴더에 '주차-#번째작업-report.md' 형식으로 만들어.
