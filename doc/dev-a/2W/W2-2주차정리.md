# [2주차-2] 입력받은 데이터 저장하기 (DB 저장)

이 문서는 환자가 전송한 정보를 서버가 어떻게 받아서 데이터베이스(DB)라는 '전자 창고'에 저장하는지 설명합니다.

---

## 🧭 1. 워크플로우(Workflow): 어떤 계획을 세웠나요?
*   **목적**: 화면에서 보낸 정보를 받아 DB에 '영구히' 저장하자!
*   **설계**: 
    1. 사용자가 보낸 데이터를 담을 '택배 상자(DTO)'인 `ReservationCreateForm`을 만들자.
    2. 컨트롤러에서 `POST /reservation/create` 주소로 데이터를 받자.
    3. 서비스(Service)와 레포지토리(Repository)를 통해 DB에 쑤욱 저장하자.

---

## 🏗️ 2. 리포트(Report): 실제로 어떻게 구현했나요?
자바 코드로 데이터를 가공하고 저장하는 과정을 살펴봅시다.

### 2.1. 데이터를 담는 택배 상자: ReservationCreateForm.java
이름과 전화번호가 적힌 폼 데이터를 자바 객체로 변환합니다.

```java
// 이 클래스는 사용자가 보낸 정보를 담아두는 틀입니다.
public record ReservationCreateForm(
    // 화면의 name="name"과 일치해야 스프링이 알아서 넣어줍니다!
    String name,
    // 화면의 name="phone"과 일치해야 합니다.
    String phone,
    // 진료과와 날짜 정보도 포함합니다.
    Long departmentId,
    LocalDate reservationDate
) {}
```

### 2.2. 주방장과 요리: ReservationService.java
실제로 데이터를 검사하고 저장하는 핵심 일꾼입니다.

```java
@Service // "나는 요리사(Service)야!"라고 명시합니다.
public class ReservationService {

    private final ReservationRepository reservationRepository;

    @Transactional // "DB 작업이니까 안전벨트 매고 시작할게!"
    public Long saveReservation(ReservationCreateForm form) {
        
        // 1. 받은 정보를 바탕으로 '예약(Reservation)'이라는 실제 객체를 만듭니다. (Build)
        Reservation reservation = Reservation.builder()
            .name(form.name())
            .phone(form.phone())
            .status(ReservationStatus.PENDING) // 처음엔 '대기' 상태로!
            .build();

        // 2. 창고지기(Repository)에게 DB 창고에 넣어달라고 합니다.
        // 저장(save)이 끝나면 DB에서 부여한 고유 번호(ID)가 생성됩니다.
        Reservation saved = reservationRepository.save(reservation);

        // 3. 나중에 확인을 위해 방금 저장된 예약 번호를 돌려줍니다.
        return saved.getId();
    }
}
```

---

## 💡 3. 왜 이 코드를 사용했고, 어떤 기능을 하나요?
*   **DTO (ReservationCreateForm)**: 사용자가 보낸 10개의 정보를 10개의 파라미터로 받으면 코드가 지저분해집니다. 이를 **한 상자에 담아서 깔끔하게 처리**하기 위해 사용합니다.
*   **Service (서비스)**: 복잡한 일(유효성 검사, 상태 설정 등)을 수행합니다. **컨트롤러가 해야 할 무거운 일**을 대신 처리해주는 비즈니스 로직의 중심입니다.
*   **`@Transactional`**: 만약 데이터를 저장하다가 서버가 꺼지면 어떻게 될까요? 이 어노테이션은 **"전부 성공하거나, 아니면 없었던 일로 하거나"**를 보장해 주는 보안 기능입니다.
