# W3-3 리포트 - 추천 결과 영역 UI 시간 추천 항목 추가

## 작업 개요
- **작업명**: 추천 결과 영역에 시간 추천 항목 추가 (진료과·의사·시간 3항목 표시)
- **수정 파일**: `src/main/resources/templates/reservation/symptom-reservation.mustache`

## 작업 내용

### 1. HTML — rec-time 카드 추가
추천 결과 grid를 `grid-cols-2` → `grid-cols-3`으로 변경하고 시간 카드 추가.

```html
<div class="grid grid-cols-3 gap-4 mb-6">
  <div>추천 진료과 → id="rec-dept"</div>
  <div>추천 전문의 → id="rec-doctor"</div>
  <div>추천 시간   → id="rec-time"</div>  ← 신규 추가
</div>
```

> **💡 입문자 설명**
> - **이 코드가 하는 일**: 추천 결과를 3칸 격자(grid) 형태로 보여주는 HTML 구조입니다. 진료과·전문의·시간 세 항목이 가로로 나란히 표시됩니다. 기존 2칸(`grid-cols-2`)에서 3칸(`grid-cols-3`)으로 확장했습니다.
> - **왜 이렇게 썼는지**: Tailwind CSS의 `grid-cols-3`은 자식 요소를 3열로 균등하게 배치하는 클래스입니다. `gap-4`는 각 칸 사이의 간격을 설정합니다. 각 `div`에 고유한 `id`를 부여해서 JS에서 찾아 내용을 채울 수 있습니다.
> - **쉽게 말하면**: 결과를 보여주는 표의 열 개수를 2개에서 3개로 늘린 것입니다.

### 2. JS — SYMPTOM_MAP 및 DEFAULT_RESULT에 time 필드 추가

```js
const SYMPTOM_MAP = [
  { keywords: ['열', '기침', '감기', '콧물', '목', '인후'], dept: '내과',       doctor: '김민준 원장', time: '09:00' },
  { keywords: ['수술', '상처', '외상', '찢김', '골절', '뼈'], dept: '외과',     doctor: '이성호 원장', time: '10:00' },
  { keywords: ['아이', '어린이', '소아', '유아'],             dept: '소아과',    doctor: '박지수 원장', time: '11:00' },
  { keywords: ['귀', '코막힘', '이비인후', '코', '귀통증'],  dept: '이비인후과', doctor: '최동우 원장', time: '14:00' },
];
const DEFAULT_RESULT = { dept: '내과', doctor: '김민준 원장', time: '09:00' };
```

> **💡 입문자 설명**
> - **이 코드가 하는 일**: `time` 필드가 추가된 최종 `SYMPTOM_MAP`입니다. 각 진료과별 더미 추천 시간이 설정되어 있습니다. 기본값도 time 필드를 포함합니다.
> - **왜 이렇게 썼는지**: 각 진료과마다 서로 다른 시간대(09:00, 10:00, 11:00, 14:00)를 배정해 더미 데이터임에도 현실감 있게 보이도록 했습니다.
> - **쉽게 말하면**: 각 진료과 창구마다 "담당 시간표"를 붙여놓은 것입니다.

### 3. JS — callSymptomApi 결과에서 rec-time DOM 업데이트

```js
document.getElementById('rec-time').textContent = result.time;
```

> **💡 입문자 설명**
> - **이 코드가 하는 일**: 분석 결과의 추천 시간(`result.time`)을 화면의 `rec-time` 요소에 표시합니다.
> - **왜 이렇게 썼는지**: 한 줄로 간결하게 특정 요소의 텍스트만 업데이트합니다. 진료과·의사 업데이트와 동일한 패턴을 유지해 코드 일관성을 높입니다.
> - **쉽게 말하면**: 결과 카드의 "추천 시간" 칸에 시간을 적어 넣는 것입니다.
