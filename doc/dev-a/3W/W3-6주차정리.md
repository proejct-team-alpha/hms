# [3주차-6] UI의 완성 (면책 고지 문구와 마무리)

이 문서는 AI 추천 결과가 절대적인 진단이 아님을 사용자에게 명확히 알리는 '면책 고지 문구'를 추가하여, 시스템의 신뢰도와 법적 안전성을 확보하는 과정을 설명합니다.

---

## 🧭 1. 워크플로우(Workflow): 어떤 계획을 세웠나요?
*   **목적**: 사용자가 AI 추천 결과를 '의사의 최종 진단'으로 오해하지 않도록 안내 문구를 추가하자!
*   **설계**: 
    1. 분석 결과창 하단에 "이 정보는 참고용입니다"라는 작은 안내 글씨를 넣자.
    2. 눈에 너무 띄지 않으면서도 읽을 수 있는 크기(text-xs)와 색상(text-slate-400)으로 디자인하자.
    3. 정보성 아이콘(Info)을 곁들여 한눈에 알림임을 알 수 있게 하자.

---

## 🏗️ 2. 리포트(Report): 실제로 어떻게 구현했나요?
디테일이 살아있는 HTML 코드를 살펴봅시다.

### 2.1. 안내의 정석: 면책 고지 HTML
추천 결과 영역 가장 아래쪽에 위치한 안내 문구입니다.

```html
<!-- text-xs: 가장 작은 글자 크기를 사용하여 부차적인 정보임을 나타냅니다. -->
<!-- text-slate-400: 은은한 회색 계열을 써서 시각적 피로도를 낮춥니다. -->
<p class="mt-4 text-xs text-slate-400 text-center flex items-center justify-center gap-1">
  
  <!-- data-feather="info": 정보(Info)를 뜻하는 작은 아이콘입니다. -->
  <i data-feather="info" class="w-3 h-3"></i>
  
  <!-- 💡 핵심 메시지: 이 정보는 '의학적 진단'이 아니라는 점을 분명히 합니다. -->
  AI 추천은 참고용이며, 의학적 진단이 아닙니다.
</p>
```

---

## 💡 3. 왜 이 코드를 사용했고, 어떤 기능을 하나요?
*   **Flex Box (`flex`, `justify-center`)**: 아이콘과 글자를 가로로 나란히 배치하고, 화면 중앙에 **완벽하게 정렬**하기 위해 사용합니다.
*   **유틸리티 클래스 (`text-xs`, `text-slate-400`)**: CSS 파일을 따로 만들지 않고도 **태그 안에서 즉석으로 스타일을 지정**할 수 있게 돕습니다. (Tailwind CSS 방식)
*   **윤리적 가이드라인**: 기술적으로 뛰어나기만 한 것이 아니라, **사용자가 정보를 올바르게 소비하도록 돕는** 서비스 기획의 정수를 담고 있습니다.
